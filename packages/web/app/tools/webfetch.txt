Fetches content directly from a specific URL. Use this tool when you have an exact URL (http:// or https://) that you want to fetch and analyze.

IMPORTANT: 
- Use webfetch when you have a SPECIFIC URL to fetch (e.g., "https://example.com/page", "http://api.example.com/data")
- Use websearch when you need to SEARCH for information with keywords/queries (e.g., "latest React features", "Python tutorials")
- Do NOT use websearch when you already have the URL - use webfetch instead

Usage:
- The URL must be a fully-formed valid URL starting with http:// or https://
- Format options: "markdown" (default), "text", or "html"
- Markdown format converts HTML to readable markdown
- Text format extracts plain text from HTML, removing scripts and styles
- HTML format returns the raw HTML content
- Results may be summarized if the content is very large
- Default timeout is 30 seconds (max 120 seconds)

Note: This tool performs real network requests via the browser fetch API. Use this tool when you need to retrieve and analyze content from a specific URL.
