Search the web using Exa AI - performs real-time web searches based on search queries. Provides up-to-date information for current events and recent data.

IMPORTANT: This tool is for SEARCH QUERIES only (e.g., "latest React 19 features", "Python async best practices"). Do NOT use this tool when you have a specific URL to fetch - use the webfetch tool instead for direct URL access.

Usage:
- Provide a search query (keywords, questions, topics) to find relevant web results
- Supports configurable result counts (default: 8)
- Search types: "auto" (balanced, default), "fast" (quick results), "deep" (comprehensive search)
- Live crawling modes: "fallback" (backup if cached unavailable) or "preferred" (prioritize live crawling)
- Configurable context length for optimal LLM integration (default: 10000 characters)
- Returns relevant content from the most relevant websites
- Use this tool for accessing information beyond knowledge cutoff when you need to SEARCH for information
- Use webfetch tool when you have a specific URL (http:// or https://) to fetch directly
